GPU Event

Menu Path : Context > GPUEvent

The GPU Event Context allows you to spawn new particles from particular Blocks in Update or Initialize Contexts.

Context settings

Settings Type Description
Evt GPUEvent Connection from a Block that triggers a GPU Event. The Block that trigger a GPU Event are:
• Trigger Event Always.
• Trigger Event On Die.
• Trigger Event Rate

Flow

Port Description
SpawnEvent Connection to an Initialize Context. Flow anchors from a Spawn Context and a GPU Event cannot mix.
