Get Attribute: mass

Menu Path : Operator > Attribute > Get Attribute: mass

The Get Attribute: mass returns the mass, which is a standard attribute, of a simulated element depending on Location. This Operator outputs the mass of the particle in kg/dm3.

Operator settings

Setting Type Description
Location Enum The location of the attribute. The options are:
• Current: Gets the value of the attribute from the current system data container. For example, particle data from a Particle System.
• Source: Gets the value of the attribute from the previous system data container read from. You can only read from this Location in the first Context of a system after a system data change. For example, in an Initialize Particle Context.

Operator properties

Output Type Description
mass float The value of the mass attribute, based on Location.
If this attribute has not been written to, this Operator returns the default attribute value.

Details

The value the attribute returns uses the system’s space (either local-space or world-space).
